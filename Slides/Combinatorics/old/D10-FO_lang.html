1. Multisets - First order logic

In First-order logic even sets are not first-class citizens.
Unary predicates, e.g. $worker(Alice),\dots,worker(Nick)$, can group constants into sets, but we cannot assign sets to variables.
Constants are always distinguishable: no multisets. We can only use predicates/functions to map each constant to the number of copies.
$letter(B,1), letter(N,2), letter(A,3)$
But this requires explicit extra logic rules to manipulate this information.
