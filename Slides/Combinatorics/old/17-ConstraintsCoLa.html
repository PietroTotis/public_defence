3. Constraints

In CoLa there are 3 types of constraints:
  • Size constraints - define the size of multisets and configurations.
    labelled property workers; #workers = 14;
    groups in {{workers}};
    #groups = 3;
    #(#part = 7) = 1; #(#part = 5) = 1; #(#part = 2) = 1;
  • Counting constraints - specify the number of objects (sets) with a property in a configuration.
    labelled property tvs; #tvs = 12;
    property defective; #(tvs & defective) = 3;
    purchase in {tvs}; #purchase = 5;
    #(purchase & defective) >= 2;
