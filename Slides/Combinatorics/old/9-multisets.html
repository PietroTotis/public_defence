1. Multisets

A multiset is a pair $(S,f)$ where $S$ is a set and $f:S\rightarrow\mathbb{N}$ is a function counting the identical copies of each element.

$\{B, A, N, A, N, A\} = (\{B,A,N\},f), f(B)=1, f(A)=3, f(N)=2.$

A language with multisets as firs-class citizens allows us to use them as arguments and assign them to variables.
