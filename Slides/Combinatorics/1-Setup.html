1. Combinatorial Problem Solving

Problem
Compute the solution of a combinatorics math word problem: given a set of objects ﬁnd the number of valid conﬁgurations deﬁned.
Example
"A shipment of 12 diﬀerent TVs contains 3 defective ones. In how many ways can a hotel purchase 5 of these TVs and receive at least 2 of the defective ones?"
Output
The count of valid conﬁgurations. Example: 288 possible TV purchases.
