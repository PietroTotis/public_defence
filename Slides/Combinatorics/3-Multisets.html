1. Multisets

In a set all elements are distinguishable (unique label).

    labelled property tvs;
    property defective;
    #tvs = 12;
    #(tvs & defective) = 3;
In a multiset elements can have identical copies.

    property tvs;
    property defective;
    #tvs = 12;
    #(tvs & defective) = 3;
